用一個迷你手柄實現四個功能

前日購入 8BitDo Micro 迷你藍牙手柄一隻。價格近 100 元，但有專用的精英軟件可修改鍵位配置，八位堂的建造品質也比較令人放心。不過到手才發現他們家所謂的精英軟件是一對一的，並沒有集成，本手柄的軟件只有移動端的版本，有些敷衍。

如果手頭拮据，我更推薦嘗試 20 元左右的 shanwan 牌迷你手柄，除了上述兩點外，據評論稱與八位堂不相上下。而如果使用 Joy-Con，需要用 JoyToKey 做多一層映射，而且會有降低壽命的隱患，不介意者可以嘗試。

以下的操作，都建立在我慣用手和豎握手柄習慣的基礎上，可以根據實際情況增減按鍵做調節。

安卓 Perfect Viewer 漫畫閱讀器翻頁

  1. 按說明書連接設備與手柄。
  2. 打開 Perfect Viewer 進入書架頁 -> 左上角側邊欄 -> 設定 -> 控制及硬體按鍵 -> 拉到最底下點擊管理硬體按鍵
  3. 按動手柄按鍵即可自定義按鍵功能，需要注意的是
    1. 默認的按鍵碼可能會與手柄的按鍵碼衝突，可以右滑刪除它們再添加。
    2. 需要填入說明。

  2. 打開 Anki 打開左上角側邊欄 -> 設定 -> 控制 -> Answer buttons -> 新增按鍵 -> 按動手柄按鍵 -> 問與答

文石電子書閱讀器翻頁

  1. 按說明書連接設備與手柄。
  2. 用 USB 線連接閱讀器與電腦。下載 KeyMapper 並傳輸到任一個文件夾（最好便於確認，如 .*\Poke5\內部共用儲存空間\Download\AppMarket 的文件會出現在閱讀器內的下載文件夾中）。
  3. 在閱讀器上進入文件頁，找到安裝包並安裝，接着可以到應用頁找到位於末尾的 KeyMapper 並打開。
  4. 授予權限：
    1. 其中包括用該應用替代內置鍵盤的權限，因此在使用映射功能時無法使用內置鍵盤。
    2. 沒有 ROOT 的一般情況下，需要在電腦上配置 ADB，接着按照文檔賦予 WRITE_SECURE_SETTINGS 權限。
  5. 設定映射：
    1. 點擊 RECORD TRIGGER 並在五秒內按下需要的按鍵，再點擊ADD ACTION 從 KEY CODE 一欄中選擇需要的按鍵。
    2. 在默認閱讀器中，有三種翻頁方式：
      1. 21-22 DPAD，映射到手柄的默認翻頁按鍵，即左右十字鍵進行翻頁。
      2. 24-25 VOLUME，映射到手柄的音量鍵進行翻頁。經過測試，此映射延遲最低，推薦。
      3. 92-93 PAGE，映射到翻頁鍵進行翻頁。
    3. 設定好 Actions 後點擊該動作後的 ⋮，關閉 Repeat actions，另一個 “toast message” 開關可以在發現映射無法生效時開啓，以查看軟件是否已經接受到輸入，以此開始排錯。

安卓串流電腦文字遊戲時翻頁

  1. 手機上安裝精英軟件進行鍵位映射，由於 GalGame 的鍵位各有不同，因此需要根據遊戲做調整。
  2. 配置 moonlight+sunshine 串流。
  3. 手柄連接電腦，打開 Galgame 後可以直接使用手柄進行操作。
